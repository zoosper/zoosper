# Zoosper Media documentation

This folder is the long-term documentation home for the `zoosper/media` package.

Root project docs should link here instead of duplicating package-specific detail once `zoosper/media` is extracted into its own repository.

## Sections

```text
docs/architecture/
docs/operations/
```

## Root docs policy

Root `docs/` remains the project index, website source, cross-package architecture map and release roadmap. Package-specific implementation detail should live with the package so AI review tools and standalone package repositories can load less unrelated context.
