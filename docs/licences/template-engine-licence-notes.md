# Template engine licence notes

## Latte

Latte should be added as a Composer dependency in a future phase rather than copied into Zoosper source. Keep upstream licence files/notices intact and review compatibility with Zoosper's chosen project licence before public stable release.

## Twig

Twig can be supported later through an adapter if required by developers/themes.

## Rule

Third-party template engines should be integrated through Composer and service-provider adapters, not vendored manually into core source.
